# pdfkinfe

pdf toolkit to merge, split, compress, etc

**merge pdf**

```
usage: mergepdf.py mypdf1.pdf mypdf2.pdf [-o output.pdf]
   or  mergepdf.py mypdf1.pdf mypdf2.pdf:1-200 [-o output.pdf]
```
